## Usage

1. Browse products on the main page
2. Use the menu to filter by category (All, Dairy, Bakery, Fruits, Grains, Meat, Vegetables)
3. Click "Add to Cart" to add items
4. View your cart at the bottom
5. Click "Checkout" to complete your order
6. Click "Clear Cart" to remove all items
